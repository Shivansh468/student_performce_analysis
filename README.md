📊 Student Performance Analysis using Exploratory Data Analysis (EDA)
📖 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Students Performance in Exams dataset from Kaggle. The objective is to understand the factors influencing students' academic performance using Python and various data visualization techniques.

The project was developed in Google Colab using the Python data science ecosystem.
🎯 Objectives

    Explore and understand the dataset.
    Clean and preprocess the data.
    Perform Univariate, Bivariate, and Multivariate Analysis.
    Identify relationships between different student attributes.
    Visualize important trends and patterns.
    Draw meaningful insights from the data.

🛠️ Technologies Used

    Google Colab
    Pandas
    NumPy
    Matplotlib
    Seaborn

📈 Key Insights

    Female students generally perform better in Reading and Writing.
    Students completing the test preparation course score higher.
    Reading and Writing scores have a strong positive correlation.
    Students with standard lunch generally perform better.
    Parental education shows a positive relationship with academic performance.
